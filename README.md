# 🏥 Health Data Analysis (Python)

This project focuses on analyzing health-related data to understand trends, compare state-wise performance, and apply statistical techniques for better decision-making.

---

## 📊 Project Overview

The dataset contains information about:

* Year
* State
* Topic & Indicator
* Health Values
* Category & Subgroup

Using Python, the data is cleaned, analyzed, and visualized to extract meaningful insights.

---

## 🚀 Features

* Data loading and preprocessing using Pandas
* Exploratory Data Analysis (EDA)
* State-wise comparison using bar charts
* Trend analysis using line graphs
* Statistical testing (T-test & Z-test)
* Linear regression for future prediction

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* SciPy
* Statsmodels
* Scikit-learn

---

## 📈 Output

* Clear visualization of health indicators across states
* Trend analysis over different years
* Statistical comparison between categories
* Prediction of future health values

---

## 📂 Dataset

The dataset includes structured health data with attributes like year, state, indicator, value, category, and subgroup.

---

## ▶️ How to Run

1. Install required libraries:

```
pip install pandas matplotlib scipy statsmodels scikit-learn openpyxl
```

2. Run the Python file:

```
python python tool box project.py
```
